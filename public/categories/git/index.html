Git notes

Git cheat sheet for everyday life. Will update this in future… $ git init # transform the current directory into a Git repository. This adds a .git folder to the current directory and makes it possible to start recording revisions $ git config --global user.email "you@example.com" $ git config --global user.name "Your Name" $ git config --global color.status auto $ git config --global color.diff auto $ git config --global color.branch auto $ git clone <git host>:/repo/<project name>. »
