Least common multiple (LCM)

Least Common Multiple (LCM) of two number a and b is the smallest positive integer that is divisible by both a and b. Since division of integers by zero is undefined, this definition has meaning only if a and b are both different from zero. int lcm(int a, int b) { for ( int i = 1; ; i ) { if ( i % a == 0 && i % b == 0 ) { return i; } } } There is an another technique for LCM finding called Reduction by the greatest common divisor »
